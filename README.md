## Steps

### Solidity Project

#### Install Foundry
Recommended way is using `foundryup`: https://book.getfoundry.sh/getting-started/installation#using-foundryup

#### Install dependencies

```shell
$ forge install
```

#### Build project
```shell
$ forge build
```

### Rust Project

#### Install
```shell
$ cargo install
```

#### Run
```shell
$ cargo run -- --nocapture
```
